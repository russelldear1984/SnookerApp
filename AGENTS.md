# AGENTS.md

## Project rules
This repository contains a GitHub Pages deployed Next.js app.

## Important constraints
Do not modify deployment or repository configuration unless explicitly asked.

Do not change these files unless the task specifically requires it:
- .github/workflows/*
- next.config.mjs
- package.json
- README.md

## Preferred scope for normal tasks
Focus changes on:
- app/*
- components/*
- lib/*
- state/*
- styles/*
- other application UI/business-logic files

## Product direction
- Keep the app simple
- Only one active tournament at a time
- Show standings for the active tournament only
- Bookings should display player/team names
- Knockout progression should update from results
